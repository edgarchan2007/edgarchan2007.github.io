Unit 1 Vocabulary

Bit

A binary digit, either 0 or 1.

Byte

A sequence of 8 bits.

Roundoff

Error when bits aren’t enough to represent a number precisely.

Analog Data

Values that change smoothly over time, like pitch or volume.

Lossless

Compression that preserves all data and allows full recovery.

Lossy

Compression that discards some data and prevents full recovery.

Metadata

Data about data, such as info about a file.

Unit 2 Vocabulary

Computing Device

A physical device that can run a program (computer, phone, sensor).

Computer Network

A group of interconnected devices that send/receive data.

Path

The series of connections from sender to receiver.

Protocol

A set of rules for system communication.

Fault Tolerant

A system that keeps working even if parts fail.

Scalability

The ability of a system to expand or shrink with demand.

Bandwidth

The maximum transmission capacity (bits per second).

URL

An easy-to-remember web address (e.g., www.example.com).

Physical Network

The hardware and physical connections carrying signals.

IP

Assigns unique IP addresses and routes data.

TCP

Ensures reliable, ordered, error-checked delivery.

UDP

Faster than TCP but with minimal error-checking.

Packets

Small chunks of data used for transmission.

Routers

Specialized computers that forward data along paths.

DNS

Translates domain names into IP addresses.

HTTP

Protocol for transmitting web pages and media.

WWW

A system of linked pages/media browsable via HTTP.
