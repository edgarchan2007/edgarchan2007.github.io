Unit 1 Vocabulary

Bit

A binary digit, either 0 or 1.

Byte

A sequence of 8 bits.

Roundoff

Error from not enough bits to represent a number with full precision.

Analog Data

Values that change smoothly, like pitch or volume.

Lossless

Compression that preserves all original data.

Lossy

Compression that discards some data permanently.

Metadata

Data about data, like file information.

Unit 2 Vocabulary

Computing Device

A physical device that can run a program.

Computer Network

A group of devices that can send/receive data.

Path

The series of connections between devices in a network.

Protocol

Rules that specify how systems communicate.

Fault Tolerant

System can function even if parts fail.

Scalability

Ability of a system to expand or contract to meet demand.

Bandwidth

Max transmission capacity, measured in bit rate.

URL

A human-readable web address.

Packets

Small chunks of data messages are split into for transfer.

Routers

Specialized computers that forward data across networks.

DNS

Translates domain names to IP addresses.

HTTP

Used for transmitting and requesting web pages/media.
